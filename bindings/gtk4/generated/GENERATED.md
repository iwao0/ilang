# generated GTK 4 bindings

The `<ns>.il` files in this directory are emitted by `gtk4-bindgen` from the GObject Introspection metadata (`/opt/homebrew/share/gir-1.0/Gtk-4.0.gir` on macOS, the distro equivalent on Linux). Do not hand-edit; rerun `cargo run -p gtk4-bindgen -- --gir ... --out bindings/gtk4` to refresh.
